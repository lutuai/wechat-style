# WechatStyle - 微信公众号排版工具

> 让排版变简单 | WechatStyle - Make Formatting Beautiful

## 🎯 项目简介

**WechatStyle** 是一款简单易用的微信公众号排版工具，帮助内容创作者快速生成专业、美观的文章排版。

### 核心功能

- ✅ **Markdown编辑器**：支持Markdown语法输入
- ✅ **实时预览**：左侧编辑，右侧实时预览效果
- ✅ **5种精美模板**：极简风、商务风、活泼风、高级感、技术风
- ✅ **样式自定义**：字号、行高、配色自由调整
- ✅ **一键复制**：复制HTML到微信编辑器

## 🎨 模板说明

### 1. 极简风 (Minimal)
- **特点**：简洁大方，留白充足
- **适用**：技术文章、知识分享
- **配色**：黑白灰为主，蓝色点缀

### 2. 商务风 (Business)
- **特点**：专业稳重，层次分明
- **适用**：企业资讯、行业分析
- **配色**：深蓝为主，灰色搭配

### 3. 活泼风 (Lively)
- **特点**：色彩鲜艳，活力四射
- **适用**：生活类、娱乐类内容
- **配色**：橙色系，温暖明亮

### 4. 高级感 (Premium)
- **特点**：精致优雅，品质卓越
- **适用**：深度文章、观点评论
- **配色**：紫色系，高贵典雅

### 5. 技术风 (Tech)
- **特点**：深色模式，极客范儿
- **适用**：技术教程、代码示例
- **配色**：黑色背景，青色高亮

## 💡 使用说明

### 基础用法

1. **输入内容**：在左侧编辑器中输入Markdown文本
2. **选择模板**：点击顶部模板按钮切换不同风格
3. **调整样式**：使用底部样式面板调整字号、行高、配色
4. **复制发布**：点击"复制HTML"按钮，粘贴到微信编辑器

### Markdown支持

- ✅ 标题（# ## ###）
- ✅ 段落和换行
- ✅ **粗体**和*斜体*
- ✅ 列表（有序/无序）
- ✅ 引用（>）
- ✅ 代码（`行内代码`和```代码块```）
- ✅ 链接和图片
- ✅ 表格（GFM）
